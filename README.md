Pirate game
=====================
Here is a description of this game on [wikipedia].

How to distribute the coins if you're the 5th pirate?

This is a solution for *n* pirates.

[Interesting situation] when *n*>200.

- Input: the number of pirates
- Output: n, n-1, ... , 1
